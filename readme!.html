Welcome To Color Game

The Color Game purpose is trying to find the correct color according to the given RGB color on the header.
RGB is representing Red, Green, and Blue. The color scale is from 0 to 255. The combination of 3 colors is resulted an unique color.
For example:

  • RGB(0, 0, 0) is representing Black
  • RGB(255, 0, 0) is representing pure Red
  • RGB(0, 255, 0) is representing pure Green
  • RGB(0, 0, 255) is representing pure Blue
  • RGB(255, 255, 255) is representing White
It should be noted that colors are changed with respect to visualization device. That is, the color you see on the screen cannot be the exact color due to device graphical system. Some hue - saturation adjustments are affecting the color that you see.
